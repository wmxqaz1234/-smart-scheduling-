# 每周智能排班

自动为下周一~周日生成排班表，写入简道云排班结果表。

## 执行

使用 bash 执行以下命令（一条命令完成全部逻辑）：

```bash
python skills/smart-scheduling/scripts/run_weekly_schedule.py --week-offset 1
```

## 输出

脚本会自动完成：
1. 从简道云读取在职员工、班次模板、排班规则
2. 运行排班算法（约束求解 + 公平性优化）
3. 将排班结果批量写入简道云排班结果表（关联字段使用 data_id 格式）
4. 输出排班摘要（填充率、人均天数、公平性指标）

## 异常处理

- 如果脚本报错"配置不存在"，需要先运行 init_config.py
- 如果脚本报错"无在职员工"，需要先在简道云员工信息表录入数据
- 如果排班填充率低于 80%，考虑增加员工或调整班次需求
